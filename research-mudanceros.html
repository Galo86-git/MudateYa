Trae todas las empresas de la zona desde Google Maps con rating, reviews, teléfono y dirección. Dos modos: buscar por nombre (lista cerrada) o descubrir por zona (lista completa).

1 Pegá el token de admin
Este HTML llama al endpoint /api/cotizaciones?action=places-research que usa la Google Places API key guardada en Vercel. Tu API key nunca se expone en el browser.
2 Elegí cómo buscar
El campo después del | es lo que Google va a buscar. Cuanto más específico, mejor el match. Si no ponés |, se usa el nombre tal cual.
Google devuelve hasta 60 resultados por consulta (3 páginas de 20). Si ponés varias líneas, hace una búsqueda por cada una y mergea los resultados sin duplicar. Cuantas más queries, más empresas vas a encontrar — y más calls a la API consume.
